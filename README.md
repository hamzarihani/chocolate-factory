# Maison du Chocolat | Artisanal Chocolate Factory

Welcome to **Maison du Chocolat**, where tradition meets artistry. This project is a digital showcase for an artisanal chocolate factory founded in 1897, featuring handcrafted luxury chocolates made with the finest single-origin cacao.

## ✨ Features

- **Artisanal Collection**: Explore our curated selection of truffles, bars, and specialty chocolate creations.
- **Multilingual Support**: Fully localized experience in **English**, **French**, and **Arabic** (including RTL support).
- **Admin Dashboard**: Secure management interface for products and profile settings.
- **Responsive & Modern UI**: Built with a focus on luxury aesthetics using Next.js, Tailwind CSS, and Radix UI.
- **Performance Optimized**: Leveraging Next.js 15+ features for fast loading and smooth transitions.

## 🚀 Technologies

- **Framework**: [Next.js](https://nextjs.org/)
- **Library**: [React](https://reactjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [Radix UI](https://www.radix-ui.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Internationalization**: Custom i18n implementation with dictionary support.
- **Analytics**: [Vercel Analytics](https://vercel.com/analytics)

## 🛠️ Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/chocolat-factory.git
   cd chocolat-factory
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open the application:**
   Navigate to `http://localhost:3000` to see the result.

## 📂 Project Structure

- `app/`: Next.js App Router pages and layouts.
- `components/`: Reusable UI components.
- `dictionaries/`: JSON translation files for i18n.
- `hooks/`: Custom React hooks.
- `lib/`: Utility functions, contexts, and mock database.
- `public/`: Static assets (images, icons).
